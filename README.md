# Indian Pincode Boundary Mapping

State-wise and district-wise Indian PIN code boundary visualization using GeoPandas.

## Overview

This project visualizes Indian PIN code boundaries using geospatial data and Python. It combines PIN code boundary polygons with latitude-longitude information to generate state-level and district-level maps for analysis and research.

The notebook enables users to explore postal regions, visualize PIN code coverage, and export structured datasets for further study.

## Features

* State-wise PIN code boundary visualization
* District-wise PIN code analysis
* Color-coded geospatial maps
* PIN code indexing within districts
* CSV export of processed data
* GeoJSON and coordinate data integration

## Workflow

1. Upload the required GeoJSON and CSV files.
2. Load and merge boundary and coordinate datasets.
3. Select a state for state-level visualization.
4. Select a district for detailed district-level analysis.
5. Generate maps and export results as CSV files.
